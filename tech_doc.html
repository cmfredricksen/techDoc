Initializing Git to start tracking your files.

Git is version control software. It keeps track of every change you make to your files. This allows you the freedom to go back to an earlier rendition of your code if things go awry. Another handy feature is the ability to work on branches, only merging to your main project once everything is working properly.

In order for Git to track your changes you need to start a repository for your file(s). You will do this by using the 'git init' command in your terminal. Git init is short for git initialize, and this command begins a repository for your file.
